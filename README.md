> **Plataforma corporativa de intermediação de serviços com interface Glassmorphism e filtragem dinâmica de alta performance.**

Este projeto é uma aplicação web desenvolvida para conectar clientes a profissionais qualificados de forma rápida e segura. A interface utiliza conceitos modernos de design para garantir retenção e uma experiência de usuário (UX) impecável, com foco total em conversão e usabilidade técnica.

---

## 🚀 Funcionalidades

### ⚙️ Motor de Busca e Filtragem Dinâmica
* **Real-time Filtering:** Sistema desenvolvido em Vanilla JS que filtra a lista de profissionais instantaneamente conforme a digitação. 
* **Algoritmo Inteligente:** O código varre títulos (`h3`) e descrições (`p`) no DOM, oferecendo feedback visual imediato sem recarregar a página.
* **Persistência de Dados:** Integração entre a Home e a página de resultados via `localStorage`. 
* **Sincronização:** O termo buscado na página inicial é processado e aplicado automaticamente ao carregar a lista de profissionais na página de busca.

### 💎 UI/UX e Estilização Avançada
* **Glassmorphism Design:** Implementação de camadas de "vidro" (blur e transparência) via CSS3 puro, garantindo um visual premium que se destaca no mercado de prestação de serviços.
* **Navegação Padronizada:** Header e Footer consistentes em todas as interfaces (Home, Login, Cadastro e Busca), utilizando um sistema de rotas simples para um ecossistema completo.

### 🔐 Fluxo de Autenticação e Conversão
* **Smart Forms:** Telas de Login, Cadastro e Recuperação de Senha com validações em tempo real via JavaScript, focando em reduzir a fricção no momento do cadastro.
* **Responsive Architecture:** Layout 100% responsivo, garantindo que o cliente consiga contratar um serviço com a mesma facilidade tanto no Desktop quanto no Mobile.

---

## 🛠️ Tecnologias Utilizadas

* **Frontend:** HTML5 Semântico, CSS3 (Flexbox & Grid).
* **Linguagem:** JavaScript (ES6+) "Vanilla" — foco em performance pura e zero dependência de frameworks pesados.
* **Design System:** Glassmorphism, Gradientes dinâmicos e variáveis CSS para manutenção simplificada.
* **Tipografia:** Família de fontes *Inter* via Google Fonts.
