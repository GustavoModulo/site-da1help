# 🚀 Dá1Help - Plataforma de Conexão de Serviços

O *Dá1Help* é uma solução tecnológica desenvolvida para intermediar a conexão entre profissionais autônomos qualificados e clientes. Este repositório contém o front-end oficial da plataforma, focado em alta performance e design moderno.

## 🏢 Sobre a Empresa
A Dá1Help foca em transformar o mercado de serviços através de uma interface intuitiva e processos transparentes, garantindo que o talento certo encontre a oportunidade ideal.

## ✨ Funcionalidades
* *Filtro Inteligente*: Busca de profissionais por categoria em tempo real (JavaScript Vanilla).
* *Interface Premium: Implementação de *Glassmorphism com desfoque de fundo e gradientes.
* *Navegação Consistente*: Fluxo completo entre Home, Busca, Cadastro e Login.
* *Design Responsivo*: Experiência otimizada para dispositivos móveis e desktop.

## 🛠️ Tecnologias
* HTML5 / CSS3 (Grid e Flexbox)
* JavaScript ES6+ (Manipulação de DOM e LocalStorage)
* Google Fonts (Inter)

## 📂 Organização do Repositório
Para manter a integridade visual, a estrutura de pastas é:
* /images: Assets e fotos de perfil (Caminho base: ../images/).
* /Mainpages: Arquivos de lógica e estrutura HTML.
